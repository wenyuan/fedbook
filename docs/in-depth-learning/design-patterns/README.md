# 设计模式

**为什么要学习设计模式（Why）**：

* 我本科读的非科班，靠自学入的行，因此这块知识需要补齐。
* 对于项目技术负责人来说，设计能力是必要基础。
* 从写好代码，到做好设计，设计模式是必经之路。
* 现在很多框架的源码中大量运用了各种设计模式，想要读懂框架源码，这是前置条件。
* 中高级开发工程师的面试也会考察设计能力。

**怎么学设计模式（How）**:

一共有 5 大设计原则，23 种设计模式。
